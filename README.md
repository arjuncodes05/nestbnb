# NestBnB 🏡

NestBnB is a web application inspired by Airbnb that allows users to explore rental properties, view listings, and manage accommodation bookings.  
The project demonstrates full-stack web development concepts including authentication, property listing management, and responsive UI design.

## ✨ Features
- **User Authentication:** Secure sign-up and login for guests and hosts
- **Property Listings:** Browse available listings with images, descriptions, prices, and locations
- **Search & Filters:** Find the perfect place using search criteria like location, price range, and amenities
- **Booking System:** Request to book a property for specific dates
- **Host Dashboard:** Allow hosts to manage their properties and view bookings
- **Responsive Design:** Optimized for desktops, tablets, and mobile devices
- **Email Booking Confirmation**: Booking details share with user via email.
